# Part Studio

## Quick Actions

- Select a visible reference plane (`Top`, `Front`, or `Right`) from the project tree or the 3D preview.
- When a plane is selected, a quick actions rail appears on the right side of Part Studio.
- Use `Sketch` in that rail to start sketching on the selected plane.
- After a solid is extruded, you can also select one of its visible faces in the 3D preview and start a sketch from that face.

## Sketch Mode

- After entering sketch mode, the same right-side rail becomes the main action surface.
- Use `Line` or `Rectangle` to choose the active drawing tool.
- Use `Undo`, `Reset`, and `Finish Sketch` as the sketch changes state.
- Set `Extrude height` and use `Extrude` after the sketch is closed.
- Extruding from a selected solid face creates a new extrusion aligned to that face, and additional extrusions appear as separate entries in the part tree.
- Use `Exit Sketch` to return to the normal part view while keeping the current plane selected.

## Preview Interaction

- Click a visible reference plane in the preview to select it.
- Drag the corner rings on the selected plane to resize it.
- In sketch mode, click on the selected plane to place sketch geometry directly on that plane.
- Use the mouse wheel over the preview to zoom, and that zoom level is kept with the part state.
