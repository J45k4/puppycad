# Part Studio

## Quick Actions

- Select a visible reference plane (`Top`, `Front`, or `Right`) from the project tree or the 3D preview.
- When a plane is selected, a quick actions rail appears on the right side of Part Studio.
- Use `Sketch` in that rail to start sketching on the selected plane.
- Select an extrude face in the 3D preview to expose the same `Sketch` action for face-based sketches.

## Sketch Mode

- After entering sketch mode, the same right-side rail becomes the main action surface.
- Sketch geometry is placed directly on the selected reference plane in the 3D preview.
- Use `Line` or `Rectangle` to choose the active drawing tool.
- Use `Undo`, `Reset`, and `Finish Sketch` as the sketch changes state.
- Finish Sketch is enabled only when the active sketch resolves to a single closed profile.
- Set a positive `Extrude height` and use `Extrude` after the sketch is finished.
- Every sketch and extrude is stored as an ordered feature in the part document and appears in the project tree.
- Use `Exit Sketch` to return to the normal part view while keeping the current plane selected.
- Select an extrude feature from the project tree to edit its blind depth or delete it from the right-side rail.

## Preview Interaction

- Click a visible reference plane in the preview to select it.
- Click an extruded solid in the preview to select that feature and edit its depth from the right-side rail.
- Click an individual extrude face in the preview to select and highlight that face while keeping the parent extrude active in the rail.
- In sketch mode, click on the selected datum plane or face to place sketch geometry directly on that target.
- Use the mouse wheel over the preview to zoom, and that zoom level is kept with the part state.
- Extruded preview solids are regenerated by replaying the saved part features through `src/cad`.

## Scope

- Part Studio currently supports reference-plane sketches, face-based sketches on extruded planar faces, and blind extrudes.
- Supported sketch entities are `Line` and `Rectangle`.
- Plane resizing and constraint solving are intentionally out of scope for this editor path.
